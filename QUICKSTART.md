# 🚀 Quick Start Guide

**Get up and running with the Cryptocurrency Intelligence Platform in 5 minutes!**

## 📋 Prerequisites

- **Python 3.8+** installed on your system
- **pip** package manager
- **Terminal/Command Prompt** access

## ⚡ 3-Step Setup

### 1️⃣ Install Dependencies
```bash
pip install -r requirements.txt
```

### 2️⃣ Launch the Dashboard
```bash
python run.py --dashboard
```

### 3️⃣ Start Exploring!
Open your browser to **http://localhost:8501** and explore:
- 📈 **Market Overview** - Real-time market analysis
- 🤖 **AI Predictions** - Machine learning forecasts
- 💼 **Portfolio Optimizer** - Professional portfolio tools
- 📊 **Technical Analysis** - Advanced charting
- 🎯 **Backtesting** - Strategy performance testing
- 📱 **Real-time Trading** - Live trading interface

## 🔥 Quick Examples

### Run Demo Analysis
